# Welcome to solradm

solradm is a feature packed command line tool that helps operators manage Solr clusters and their accompanying ZooKeeper ensembles. The CLI embraces a context system that lets you work with many environments at once and switch between them effortlessly. This website documents every part of the tool and explains how to make the most out of it.

Use the navigation links above to learn about the context system or dive straight into the command reference. Each page provides extensive background material, practical examples and tips for real world operations.
