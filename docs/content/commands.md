+++

SolrADM organizes functionality into several subcommands:

- **core** – Interact with the Core API for managing individual Solr cores.
- **coll** – Manage collections, shards, and replicas through the Collections API.
- **backup** – Create and restore backups using the Replication API.
- **context** – Maintain Solr contexts and configuration profiles.
- **zoo** – Operate on ZooKeeper to edit configuration and state.
- **auth** – Manage Solr authentication credentials.

Run any subcommand with `--help` to see its available actions.
